# Random Garbled Text Generator

![GitHub last commit (branch)](https://img.shields.io/github/last-commit/KOTO28/random-garbled-text-generator/main)

English | [日本語](README_ja.md)

A web app that generates random garbled-like text

## How it works

Generating UTF-16 strings from random numbers

## Usage

1. Open [Random Garbled Text Generator](https://koto28.github.io/random-garbled-text-generator/en/)
2. Set the length
3. Click the `Generate` button
4. Random text will be output
