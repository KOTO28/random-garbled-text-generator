# ランダム文字化けジェネレーター

![GitHub last commit (branch)](https://img.shields.io/github/last-commit/KOTO28/random-garbled-text-generator/main)

[English](README.md) | 日本語

ランダムに文字化けのようなテキストを生成するWebアプリ

## 仕組み

ランダムな数値からUTF-16文字列を生成しています

## 使い方

1. [ランダム文字化けジェネレーター](https://koto28.github.io/random-garbled-text-generator/ja/) を開く
2. 文字数を設定
3. `生成` ボタンをクリック
4. ランダムなテキストが出力されます
